# DESIGN.md — Design Philosophy

## The guiding idea

Reading Companion is built on the design philosophy of Dieter Rams and Jonathan Ive. As little as possible, but exactly right. The UI recedes completely. The companion is the only thing that matters on the page.

Jonathan Ive: *"There's beauty when something works and it works intuitively."*
Dieter Rams: *"Less, but better."*

When there is no open space, the user doesn't engage. Minimalist products leave room for the user's intuition to interact. Leave that space.

---

## Dieter Rams' ten principles — applied here

| Principle | What it means for this product |
|---|---|
| **Innovative** | Pretext-powered live text evasion is genuinely new. Design must feel native to that innovation. |
| **Useful** | Every element must serve reading or companion interaction. If it doesn't, remove it. |
| **Aesthetic** | Beautiful like a well-typeset book. Considered, not flashy. |
| **Understandable** | Drag it. Click it. Paste a URL. Nothing needs a label or tooltip. |
| **Unobtrusive** | UI recedes. Content and companion are foreground. Controls appear when needed, vanish when not. |
| **Honest** | No fake materials. No decorative shadows. No elements pretending to be something they aren't. |
| **Long-lasting** | No trends. No gradients. Timeless editorial style. |
| **Thorough** | Line height matters. Letter spacing matters. The gap between companion and text matters. |
| **As little as possible** | When in doubt, remove it. Open space invites the user's intuition. |

---

## Specific UI rules

- Typography first — line height 1.6–1.8, tracked headings, considered fonts
- White space is structural — wide margins, generous padding, content breathes
- Max 3–4 colors per theme — no gradients, no rainbow
- Motion communicates state only — drag, part, track, spin — never decorative
- The companion is the hero — everything else is quiet

---

## Design consultation process

Before writing any UI code: describe the design in plain words. Wait for human approval. Only code after approval. Revise until satisfied.
